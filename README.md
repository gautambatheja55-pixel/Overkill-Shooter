# Overkill-Shooter
Classic Space Shooter
<br>
This is my first game :) made using Godot with theme as OverKill
<br>
Villain's spacecraft gets destroyed when the laser strikes it with a explosion sound.<br>
The background moves as the player goes forward.<br>
Player's spacecraft has a sound + trail<br>
tried to add a score counter ended up with nothing<br>
when the villain's spacecraft reach the blue line the game ends.  NO AI WAS USED IN ANY PART
